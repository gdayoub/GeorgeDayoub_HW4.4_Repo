# Stat184_HW_4.4
The main goal for this repository is to practice integrating homework 4.3 into GitHub.
## Overview
This project analyzes the calcium dataset using R and Quarto from the HW 4.4 assignment. The report includes data cleaning, analysis, visualizations, and written explanations, all rendered out to a PDF.
## Interesting Insight
The biggest thing I noticed was how much the level of detail in the AI prompt changed the comparison on the calcium dataset. The basic prompt gave something basic, but my more detailed prompt produced a completely different analysis and different takeaways. Prompt quality mattered way more than I expected.
