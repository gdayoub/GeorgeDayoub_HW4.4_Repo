# Project Plan

## HW 4.3 — The Analysis

### Goal
Produce a clear, reproducible Quarto using the information from HW 3.4 and 4.3 and export it as a PDF.

### What I Needed
- calcium.csv dataset
- R and the libraries I used in the QMD
- Quarto installed

### Steps
- Load calcium.csv into RStudio
- Clean and inspect the data
- Run the analysis
- Build the plots (tables, graphs, images)
- Write explanations next to each result
- Add titles, captions, and formatting
- Render the QMD to PDF
- Check the PDF actually looks right before submitting

## HW 4.4 — The Repo

### Goal
Build a public GitHub repo around HW 4.3 that actually uses branches, issues, and pull requests.

### What I Need
- GitHub account
- HW 4.3 files (QMD, PDF, dataset, images)
- README and this PLAN file

### Steps
